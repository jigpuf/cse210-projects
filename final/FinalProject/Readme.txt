This application helps you build files that hold list of different types of media and where to find them.

You will be presented with several choices to choose from.
1.  Add to active list
This will allow you to build a list that will not continue once the program closes.  You can add to this list one item at a time
2. Show active list
This will display the active list as it currently stands
3. Save list to file
This creates a file and places the active list in the file so it can be retrieved at any time even after the program closes
4. Load list from file
Moves the data from a saved file into the active list so it can be cleared or added to.
5. Clear Active List
Clears active list
6. Quit
Clears active list and closes  program
